# Debtective

Help find out todos in your application.

## Usage

Run the task:

- `bundle exec rake debtective:todo_list`

More tasks to come!

## Installation

Add this line to your application's Gemfile:

```ruby
group :development do
  gem 'debtective'
end
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install debtective
```
